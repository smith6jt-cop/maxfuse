# MaxFuse

Cross-modal single-cell data integration with spatial awareness.

MaxFuse is a Python package for integrating single-cell datasets from different modalities with no overlapping features and/or under low signal-to-noise ratio regimes.

## Installation

```bash
pip install maxfuse
```

For development:

```bash
pip install -e .
```

## Usage

```python
from maxfuse import Fusor, Mario

# Use Fusor for MaxFuse integration
# Use Mario for statistical matchability testing
```
